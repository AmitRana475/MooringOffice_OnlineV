4.2.6 LINE CROPPING

LINE CROPPING

     This page appears from LMPR > Mooring Line > Line Cropping        

 

Line Cropping : Click on the 'Add Line Cropping' button to make an entry of Line Cropping operation, which shall open the form. Form fields are explained as below -

                       a)       Select Line : Dropdown to select the available lines as per their certificate numbers.

                       b)     Outboard End in Use (A/B): View only field

                       c)       Line winch assigned: View only field

                       d)     Line Location assigned: View only field

                       e)      Cropped Date: Calendar to select the date on which cropping was done.

                       f)       Cropped Outboard End: Selection of outboard end which was cropped.

                      g)      Cropped Rope Lt: Cropped line length in metres.

                      h)     Reasons for Cropping: Dropdown list to select one or more reasons for Cropping.
