5.2 USING TRAINING CONTENT MODULE

Using Training Content module

 

  

The features of the Training content screen may be used as following :

 

                                                              i.      Add Attachment: To add any training content file. Procedure to add a file is explained in 7.3 of this manual.

                                                             ii.      View : To view the training content, depending on the content uploaded, it will open the respective appliction from your computer to display the content, eg: for Video files, it will open your default media player, for pdf files, it will open the default pdf viewer etc.

                                                           iii.      Scrolling Pages: For large number of entries in the list, system divides it to various pages. To scroll the pages, use the toggle arrows. Single arrow is for next / previous page, double arrows is for last or first page as per the direction of the arrow.
