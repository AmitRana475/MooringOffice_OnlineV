4.1 MOORING OPERATION

Mooring Operation Records

            This section is to record all available information after a mooring operation is conducted. It is required that 'All' mooring operations where any mooring ropes were used, shall be recorded in this section. The details to fill this form are given below :

 

 

  • Adding new Mooring Operation - To add a new mooring operation, click on 'LMPR' menu, then click on 'Mooring Operations' submenu which will open Mooring operation page. On this page, click on 'Add Mooring Operation' button which will open a form to fill all details.

    

 

 

  • Mooring Operation Records form fields -

·         Port Name: Type the Port name which will search the list of all ports and suggest the available names in the list below, choose the desired name to fill this field.

In case the desired Port name is not available in this list, search for 'Other' and click to select 'Other' from the list. On selection of 'Other', a text field will open underneath this field, please insert the desired name of the Port here.

·         Date & Time for All Fast : Insert the Logbook date and time for mooring 'All Fast'. To select the date, click on the calendar icon besides, which will open a calendar, click to choose the desired date.

To enter time, choose hours from the first time dropdown and minutes from the second time dropdown.

·         Date & Time for Cast Fast : Insert the Logbook date and time for mooring 'All Cast Off'. To select the date, click on the calendar icon besides, which will open a calendar, click to choose the desired date.

To enter time, choose hours from the first time dropdown and minutes from the second time dropdown.

·         Winches used for the full period : A list of all available winches with ropes warped on them will appear here. Click the checkbox to select all used winches / ropes & rope tails. Then select the direction and lead for each rope.

In case a Winch / Rope is not used for the full period from 'All Fast' to 'Cast Off', an option is given to 'Add More'. To use this option, follow below steps -

                  i)                   Ensure winches / ropes not used for complete operations shall be unchecked from the main topmost section of winches.

                 ii)                 Press 'Add More' button, which will open a subsection underneath containing list of winches / ropes which are unchecked on top. It also displays selector of date & time from - to, which should be filled for the time the selected winches / ropes were used.

                iii)               The 'Add More' button is equipped to create 4 such additional timing subsections.

·         Facility Name : This field automatically lists the available facilities upon selection of the first field 'Port Name'.

In case the 'Port Name' is selected as 'Other', then a blank text field will appear here instead of the available list of facilities. Please type the desired name of the facility.

·         Berth Name / Number : Text field to insert Berth Name or Number.

·         Berth Type : Dropdown field with available selections from various values, select the value applicable to the specific berth where the Ship was berthed.

·         Type of Mooring : Dropdown field with available selections from various values, select the value applicable to the specific berth where the Ship was berthed.

·         Drafts Arrival / Departure Fwd and Aft : Drafts of the Vessel on Arrival and Departure forward and aft in metres, respectively.

·         Depth at Berth : Field to insert the depth at berth in metres.

·         Berthing Side : Selection from dropdown - Port or Stbd.

·         Vessel Condition : Dropdown field with available selections from various values, select the value applicable to the specific berth where the Ship was berthed.

·         Ship access : Dropdown field with available selections from various values, select the value applicable to the specific berth where the Ship was berthed.

·         Range of Tide : Field to insert the maximum tidal range in metres. 

·         Avg Wind Direction : Dropdown to select from 1 to 7, as per the average direction of wind at the berth. A diagram is given besides displaying the direction selection to be used. This direction is relative to the bow of the Vessel in clockwise direction.

·         Avg Wind Speed : Average speed of winds in knots.

·         Any Squall / Gusts experienced : Dropdown option to select Yes or No.

·         Current Speed : Average speed of current in knots.

·         Berth exposed to Sea / Swell : Dropdown option to select Yes or No.

·         Surging observed : Dropdown option to select Yes or No.

·         Any affect of passing traffic : Dropdown option to select Yes or No.

·         Ship was continuously in contact with fender: Dropdown option to select Yes or No.

·         Master's Notes for this Port Call : Text box to enter Master's observations of the Port.

·         Was any rope damaged : Dropdown option to select Yes or No.

·         Recording Line Damage : Any damages to the Lines in the duration of the operation should be recorded here itself. However, in case it is missed, there is an option later also to add/delete damaged ropes in an operation. To add damage lines, follow below steps -

                 i)                   Select 'Yes' in dropdown in the mooring operation form for the filed - 'Was any line damaged?'

                ii)                 In the bottom of this page, a new button will appear 'Add Damaged Lines', click on this button, which will open a popup with form to enter the damaged line details.

               iii)               Click on 'Add Damage Line' button to open this form.

 

 

 

  • Mooring Operation Records List View  -

  • View Damage LINE: Click on this button to View the list of lines damaged in this operation. You may add additional lines from this option.
  • View Mooring Op: Click on this button to View the Mooring Operation details which were stored previously.
  • Delete: Click on this button to delete any Mooring Op record. Please note once deleted, you will not be able to retrieve this data.
