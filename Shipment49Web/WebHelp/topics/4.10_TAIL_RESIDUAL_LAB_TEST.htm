4.10 TAIL RESIDUAL LAB TEST

TAIL RESIDUAL LAB TEST

This page appears from LMPR > Rope Residual Lab Test

 

 

To add the Rope Tail Residual Strength Test report, which you have received form the testing laboratory, please follow below steps -

 

a)       Click on 'Add Rope Tail Res Strength Test' button to make an entry, which shall open the form. Form fields are explained as below -

Select Rope: Dropdown lists Rope Tail Certificate numbers to be selected as desired

Rope Type: View only field, automatically populates on selection of Select Rope field.

Manufacturer: View only field, automatically populates on selection of Select Rope field.

Rope Location: View only field, automatically populates on selection of Select Rope field.

Lab Test Date: Select the Lab test date as mentioned in the lab testing report.

Months in Service: View only field, displayed in units as 'Months', calculated as (Lab Test Date - Installed Date).

Running Hours in Service: View only field, Current running hours as being recorded automatically.  

Residual Strength as per Test Report (Percentage): Enter percentage as mentioned in the lab test report (Max 100%).

Remarks: Text box to enter your remarks, if any.

Reset: Pressing this button resets the form entries you made, after reset, reselect the Rope to fill the form again.

Save: After entering all information in the form, click on Save button to save in records.

Cancel: Pressing this button will take you back to the list.
