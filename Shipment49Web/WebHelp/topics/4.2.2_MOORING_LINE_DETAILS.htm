4.2.2 MOORING LINE DETAILS

MOORING LINES DETAILS

All Mooring Lines details shall be listed in this section.

To add any new Lines detail, click on 'Add Lines Details' button on top right side.

 

 

  • Mooring Line Details Form View -

This page appears from LMPR > Mooring Line > Mooring Line Details> Add Lines Details.

 

 

 

The options on this page are listed below with respective details -

  • Rope Type: Dropdown to select one option from the list.
  • Rope Construction: Text field to enter Line construction details.
  • Diameter (mm) : Enter Line diameter here in millimetres.
  • Length (mtrs) : Enter Line diameter here in metres.
  • MBL (T) : Enter Line MBL here in Tonnes.
  • LDBF (T) : Enter Line LDBF here in Tonnes.
  • WLL (T) : Enter Line WLL here in Tonnes.
  • Manufacturer Name: Dropdown to select one option from the list.
  • Certificate Number: Enter the Line Certificate Number here.
  • Received Date: Compulsory field, Calendar to select date of receipt of the rope. This date can only be a past date.
  • Installed Date: Non-compulsory field, Calendar to select the installation date. In case the rope is kept as non-used spare, please leave this field blank.
  • This field is connected to calculation of discard criteria, hence only apply when the rope is in actual use.
  • Rope Tagging: Dropdown to select Yes / No. Ropes are ge
  • Existing Running Hours: Start counter of Running hours, generally applicable to used existing ropes being inserted into the software. Please enter the running hours this rope has already consumed.
  • Remarks: Text box to enter remarks related to the rope.
  • Save: Press Save button to save all data entered.
