4.3.3 ASSIGN ROPE TAIL TO WINCH

ASSIGN ROPE TAIL TO WINCH

This page appears from LMPR > Mooring Rope Tail > Assign Rope Tail to Winch

The options on this page are listed below with respective details -

Active Records: List of Rope Tails assigned to winch currently

Past Records: Past records of assignment and rotation on winches.

Assign Rope Tail to Winch: Click on this button to assign rope Tail to winch, which shall open the form. Form fields are explained as below -

a)       Select Rope Tail: Dropdown to select the available rope Tails as per their certificate numbers.

b)     Outboard End in Use (A/B): Select the current outboard end as per your Ship's marking. Rope Tail ends to be marked with A and B at opposite ends. After reeling on the winch, select the end currently on outboard side.

c)      Assigned Date: Selection of date when the rope Tail was reeled on the winch.

d)     Assign Rope Tail to Winch: Dropdown to select the desired Winch number where the selected Rope Tail is currently reeled.

e)      Rope Tail location assign: View only field which appears after selection of the Winch number
