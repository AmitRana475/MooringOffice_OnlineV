4.3.2 MOORING ROPE TAIL DETAILS

MOORING ROPE TAIL DETAILS

      All Mooring Rope Tail details shall be listed in this section.

      To add any new Rope Tail detail, click on 'Add Rope Tail Details' button on top right side:

Mooring Rope Tail Details Form View -

This page appears from LMPR > Mooring Rope Tail > Mooring Rope Tail Details> Add Rope Tail Details.

 

 

The options on this page are listed below with respective details -

 

Rope Tail Type: Dropdown to select one option from the list.

Rope Tail Construction: Text field to enter rope Tail construction details.

Diameter (mm) : Enter rope Tail diameter here in millimetres.

Length (mtrs) : Enter rope Tail diameter here in metres.

MBL (T) : Enter rope Tail MBL here in Tonnes.

LDBF (T) : Enter rope Tail LDBF here in Tonnes.

WLL (T) : Enter rope Tail WLL here in Tonnes.

Manufacturer Name: Dropdown to select one option from the list.

Certificate Number: Enter the Rope Tail Certificate Number here.

Received Date: Compulsory field, Calendar to select date of receipt of the rope Tail. This date can only be a past date.

Installed Date: Non-compulsory field, Calendar to select the installation date. In case the rope Tail is kept as non-used spare, please leave this field blank.

This field is connected to calculation of discard criteria, hence only apply when the rope Tail is in actual use.

Existing Running Hours: Start counter of Running hours, generally applicable to used existing rope Tails being inserted into the software. Please enter the running hours this rope Tail has already consumed.

Remarks: Text box to enter remarks related to the rope Tail.

Save: Press Save button to save all data entered.
