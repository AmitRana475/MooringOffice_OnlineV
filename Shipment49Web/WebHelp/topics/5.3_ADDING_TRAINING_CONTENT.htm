5.3 ADDING TRAINING CONTENT

Adding Training Content

 

 

To add training content, follow below steps (screenshots below the steps):

a)       Click on 'Add Attachment' button which opens a popup.

b)      In the popup, enter 'File Name' which will be displayed in the list after adding.

c)       Now click on 'Browse' button to browse the desired file from your computer.

d)      At the time of browsing, you can change the file type extension from the dropdown given in the browse screen of your computer, to select desired file type.

e)      After choosing the file, click on 'Open' in the browse screen.

f)        The file path will be displayed in the pop-up next to browse button.

g)       Now press 'Save' from the poup to save the Training content. After saving, click on View and verify if file is opening correctly.
